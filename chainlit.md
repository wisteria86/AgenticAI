# 🌸 Welcome to Wisteria!

Hello, I am Wisteria, your autonomous systems engineer. 

I can help you build architectures, write code, search the web, and analyze complex systems. I operate natively within a secure Docker sandbox. 

### Quick Start
- **Upload Files**: Click the attachment icon (paperclip) to seamlessly ingest documents into my workspace.
- **Configure**: Use the settings icon to change my underlying LLM model or toggle Auto-Approve for code execution.
- **View Artifacts**: Any code or files I generate will automatically appear in the right-hand panel for immediate preview!
